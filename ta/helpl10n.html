If you wish to help translation, please send a e-mail to tell me (JiaYanwei <சாஃபிதிருத்தி@gmail.com>) the target language. There are two ways to add a new translation OR update an exist translation:

  1. Without forking:
    1. Translate the .po file:
      • Download https://raw.githubusercontent.com/hltj/wxMEdit/master/po/wxmedit.pot.
      • Transalte the file to the target language and rename it with ll_CC.po (e.g. zh_CN.po, it_IT.po, ja_JP.po, ta.po, etc.); OR update the file ll_CC.po.
    2. Translate the ஆவணம் files:
      • Clone the ஆவணம் repository https://github.com/wxMEdit/wxMEdit.wiki.git.
      • Translate the markdown ஆவணம் files வீடு.md & பதிவிறக்கங்கள்.md to the target language and rename them with வீடு_ll_CC.md & பதிவிறக்கங்கள்_ll_CC.md; OR update the ஆவணம் files வீடு_ll_CC.md & பதிவிறக்கங்கள்_ll_CC.md.
    3. Send the translated files to me (JiaYanwei <சாஃபிதிருத்தி மின்னஞ்சல்>).
  2. With forking:
    1. Fork சாஃபிதிருத்தி on github.
    2. Add/update the translated .po file and ஆவணம் files as above.
    3. Create a pull request.

Thank you for your translation. I will add your name/nick to the கடன்கள். And you can also give me your website/blog address, I will add the link to the your translated pages and கடன்கள் pages in all language.
