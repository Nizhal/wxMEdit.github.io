சாஃபிதிருத்தி

  • சாஃபிதிருத்தி என்பது ஒரு குறுக்கு மேடை உரை/பதினறுமஎண் திருத்தி சி++ & சாஃநிரல்பலகைகளில் எழுதப்பட்டது .
  • சாஃபிதிருத்தி என்பது பித்துதிருத்தி இன் மேம்படுத்தப்பட்ட பதிப்பாகும், இது நிறுத்தப்பட்டது.
  • சாஃபிதிருத்தி உரை/நெடுவரிசை/பதின்ம முறைகளில் கோப்புகளைத் திருத்த முடியும், மேலும் பல பயனுள்ள செயல்பாடுகளை ஆதரிக்கிறது, எ.கா. புக்மார்க், தொடரியல் சிறப்பம்சங்கள், குறியாக்கங்கள், சொல் மடிப்பு, சொல்எண்ணிக்கை மற்றும் புதுப்பிப்புகள் சரிபார்த்தல்.
  • சாஃபிதிருத்தி ஆதரவு பொதுவான குறியாக்கங்கள் (ஒபசே8/16/32, உநிஅ-8859-x, சிபி125x, கோஐ8, பி18030, பெரிய5, ...) உரை/நெடுவரிசை முறைகளில் மட்டுமின்றி பதின்ம பயன்முறையிலும் உள்ளது.
  • பிழைத் திருத்தங்கள், மேம்பாடுகள் மற்றும் மறுசீரமைப்புடன் தொடர்ந்து பராமரிக்கப்படும் உரை/பதின்ம திருத்தியை வழங்குவதே இந்தத் திட்டத்தின் நோக்கமாகும்.

Features

  • சாஃபிதிருத்தி can run under Windows/ReactOS/Wine, Linux, BSDes, macOS, OpenIndiana, other Unix-like OSes and Haiku OS.
  • Checks for updates automatically.
  • Supports bookmark.
  • Edits files in Text, Column and Hex modes.
  • In Hex-Mode, சாஃபிதிருத்தி can open large files which size is up to 32GB (INT_MAX*16).
  • Users can change the encoding of files at runtime like Web-Browsers.
  • Supports many encodings, e.g.:
    • ஒருங்குறி (UTF-8, UTF-16/32 with Little or Big Endian);
    • ISO-8859-1~16, Windows-1250~1258;
    • KOI8-R/U;
    • MS936(GBK*), GB18030, MS950(Big5*), Windows-31J(Shift-JIS*);
    • etc.
  • Supports non-BMP ஒருங்குறி Characters Such as CJK Ext-B/C/D/E, Miao Letters, Mathematical Alphanumeric Symbols, Musical Symbols, Emoji, etc.
  • If users input a character that is not supported by current encoding, this character will be converted to ஒருங்குறி escape format (only code point format ஆதரவுed currently).
  • Regular Expression search & replace.
  • Opens multiple files on single instance.
  • பல நிரலாக்க மொழிகளின் syntax-முன்னிலைபடுத்துதலை ஆதரிக்கிறது:
    • awk, C/C++, CSS, diff/patch, D, DOS Batch Script, Flash ActionScript, HTML, Java, JavaScript, JSP, Lua, Pascal, PHP, Perl, Python, Ruby, UNIX Shell Script, x86 Assembly, XML, Fortran, TeX/LaTeX, Squirrel, C#, Visual Basic, ASP(VBScript), SQL, Verilog, VHDL, FreeBASIC.
  • சாஃபிதிருத்தி can view ASCII-Art files with appropriate monospace font e.g.:
    • Courier New under Windows/Mac OS X;
    • Andale Mono, Courier, Menlo, Monaco under Mac OS X;
    • Console and Fixed under Unix-like OS;
    • DejaVu Sans Mono, Free Mono, Liberation Mono, etc.
