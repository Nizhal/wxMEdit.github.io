更動記錄

2.9.8

  1. 增加是否在十六進位編輯區域貼上為十六進位字串的更多選擇。
  2. 增加新功能：在十六進位編輯區域覆蓋貼上。
  3. 增加更多常用編碼別名並在編碼轉換對話框中顯示。
  4. 開啟首頁或下載頁時，增加本地化支援。
  5. 增加 zh_HK、zh_MO、zh_SG 和 zh 中文翻譯支援。
  6. 增加將 Windows 下設定檔案放在 %APPDATA% 目錄中的選項。
  7. 增大視窗預設大小。
  8. 手動更新檢測像自動更新一樣的非同步進行。
  9. 改進編碼分組。
  10. 修正 clang-analyzer 檢測到的問題。
  11. 修正 Windows 下除錯結束時的記憶體洩露警告。
  12. 更新 FreeBSD 封裝設定，採用 ports 方式，切換到 pkgng。
  13. 更新簡體中文翻譯。
  14. 更新繁體中文翻譯（Vincent_TW）。
  15. 更新西班牙文翻譯（Carlos Sanchez <carlos1994sanchez[at]gmail.com>）。
  16. 更新日文翻譯（Tilt <tiltstr[at]gmail.com>）。
  17. 其它小修改及問題修正。

2.9.7

  1. 增加新編碼「用於 ASCII 藝術的 CP437 變體」以更好支援 ASCII 藝術。
  2. 增加自動更新檢測功能。
  3. 增加書籤功能〔主要由 Gospodin Gyurov（即 Gogo） <ggurov[at]yahoo.com> 開發〕。
  4. 增加新命令：刪除前一字/後一字（由 Gogo開發）。
  5. 增加尋找選項「循環尋找」。
  6. 關於對話框增加致謝標籤。
  7. 選項對話框增加雜項標籤。
  8. 增加三點選擇一行功能（主要由 cfreeer <cfreeer[at]users.sf.net> 開發）。
  9. 增加儲存檔案時更新語法高亮度功能（主要由 cfreeer 開發）。
  10. 增加 FreeBASIC 語法檔案。
  11. 增加編碼分組。
  12. 增加各歷史紀錄清除支援。
  13. 增加十六進位區複製行為的更多選擇並增加到選項對話框。
  14. 增加俄文翻譯（Shamil Bikineyev <shamilbi[at]gmail.com>）。
  15. 改善 Windows 下最大化功能（由 Gogo 開發）。
  16. 改善 Windows 下系統整合功能。
  17. 改善字數統計資訊的顯示效果。
  18. 變更選擇下一字行為，同其它編輯器。
  19. 變更尋找/取代對話框通知訊息到該對話框的下方。
  20. 修正取代時不會循環尋找的問題。
  21. 修正中文版 Windows 下不能同時開啟檔案「a b」和「aÀb」的問題。
  22. 修正當 debug 版 MadEdit/wxMEdit 失去滑鼠取得時斷言錯誤的問題。
  23. 修正當正在選擇文字時從 wxMEdit/MadEdit 切出導致的錯誤行為問題。
  24. 修正當檔案從外部更新後單按編輯區導致錯誤行為的問題。
  25. 更新簡體中文、日文、西班牙文翻譯。
  26. 其它小修改及問題修正。

2.9.6

  1. 改進 Mac OS X 支援。
  2. 用 wxSmith 重做各對話框並修正對話框大小問題。
  3. 重構編碼辨識邏輯修正編碼辨識問題。
  4. 修正 Windows 下十六進位模式重繪問題。
  5. 修正泛 Unix 作業系統下當游標在編輯區最上幾行時開啟多檔案尋找結果面板當掉問題。
  6. 修正當取消檔案關聯時 wxWidgets 報錯問題。
  7. 更新 RPM、debian、FreeBSD 封裝檔案。
  8. 修正自動換行最大行長設定過大導致重啟程式當掉問題。
  9. 改善編譯設定。
  10. 更新西班牙文翻譯（Carlos Sánchez <carlos1994sanchez[at]gmail.com>）。
  11. 更新日文翻譯（Tilt <tiltstr[at]gmail.com>）。
  12. 更新簡體中文翻譯。
  13. 更新正體/繁體中文翻譯（Denir Li <denir.li[at]gmail.com>）。
  14. 其它小修改。
