Hitorial de cambios

3.2

  1. added new feature: show file size in human readable format.
  2. added a option that always use certain encoding to open files.
  3. added GTK+ 3 and Wayland support.
  4. added High DPI supoport under Windows.
  5. redisigned the icons by jerome KASPER <neon.king.fr[at]gmail.com>.
  6. changed some keyboard shortcuts, see Keyboard shortcuts change in wxMEdit 3.2.
  7. updated GB18030 support to GB18030-2022 standard.
  8. updated Unicode block descriptions for Unicode 15.0 (require ICU 72 or higher).
  9. fixed crash in text mode when a line exceed the limit of line length.
  10. fixed crash when pressing OK button in "About wxMEdit" dialog.
  11. fixed not responding when warping by window with text containing some tab characters.
  12. fixed broken triple click behavior in wxMEdit 3.1.
  13. fixed that the height of Information Window in MadEdit/wxMEdit (built with wxMSW-2.8) always reduce by 4.
  14. fixed fonts and encodings not restored after restart.
  15. fixed that cannot type characters via AltGr key.
  16. fixed cursor and selection rendering under Linux.
  17. fixed that IME candidate window doesn't follow the caret under Windows 10.
  18. added French translation by jerome KASPER <neon.king.fr[at]gmail.com>.
  19. actualizada traducción al Chino Simplificado.
  20. otros cambios menores y correcciones de errores.

3.1

  1. Añadida nueva característica matadora: Límite de palabra con caracteres Chino/Japonés/Tailandés/Lao/Khmer y Birmano etc.
  2. Añadida nueva característica matadora: Ajuste de línea cumple con el estándar Unicode® Anexo #14: Algoritmo de fin de línea Unicode.
  3. añadida una característica: alinear columna.
  4. añadido icono a la barra de herramientas & acceso directo para Introducir secuencia ordenada ordenada.
  5. cambiado patrones de marca de fin de línea.
  6. actualizada descripciones de bloques Unicode a Unicode 8.0 (requiere ICU 56 o superior).
  7. corregido límite de la palabra incorrecto con caracteres no ingleses al seleccionar palabra.
  8. corregido ajuste de línea incorrecto con caracteres no ingleses.
  9. corregida entrada de caracteres rotos con teclado numérico es wxMEdit construido con wxWidgets-3.0 GTK2.
  10. corregido comportamiento incorrecto cuando se pulsa Ctrl-Retroceso al inicio de un archivo de texto o en la selección de ancho cero en el modo de columna.
  11. corregido que no se puede seleccionar la palabra entera ajustada en diferentes líneas haciendo doble clic.
  12. corregido que el triple clic no puede seleccionar una línea entera ajustada.
  13. corregido que los archivos no se guardan en el modo hexadecimal después de reiniciar & restaurar.
  14. corregida redundancia de nueva línea cuando se copia en el modo de texto normal.
  15. actualizada traducción al Japonés por Tilt <tiltstr[at]gmail.com>.
  16. actualizada traducción al Español por Carlos Sánchez <carlos1994sanchez[at]gmail.com>.
  17. actualizada traducción al Chino Simplificado.
  18. actualizada traducción al Chino Tradicional por Vincent_TW.
  19. actualizada traducción al Ruso por Станислав Будинов <stabud[at]yandex.ru>.
  20. otros cambios menores y correcciones de errores.

3.0

  1. added new options to display/print bookmark independent of line number.
  2. added a option whether enable dot matching new-line in regular expression.
  3. added a feature that show searching message in status bar when Search/Replace Dialog is invisible.
  4. añadida una característica: Insertar secuencia ordenada.
  5. improved bookmark appearance for displaying and printing.
  6. improved Tab and EOF mark pattern, make it easy to distinguish Tab mark from space mark.
  7. fixed that there are too few characters limited in a line with certain word wrap settings.
  8. fixed that width of list items in Syntax Highlighting Settings Dialog are not updated on resizing.
  9. fixed not responding when counting/replacing-all any zero-length assertions.
  10. updated Unicode block descriptions for Unicode 7.0 (require ICU 54 or higher).
  11. added German translation by Ronny Steiner <post[at]sirsteiner.de>.
  12. added Polish translation by Adam Massalski <massaada[at]wp.pl>.
  13. actualizada traducción al Chino Simplificado.
  14. actualizada traducción al Japonés por Tilt <tiltstr[at]gmail.com>.
  15. actualizada traducción al Español por Carlos Sánchez <carlos1994sanchez[at]gmail.com>.
  16. otros cambios menores y correcciones de errores.

Ver ChangeLog para más cambios.
