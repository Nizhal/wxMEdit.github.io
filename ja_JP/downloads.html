更新履歴

3.2

  1. added new feature: show file size in human readable format.
  2. added a option that always use certain encoding to open files.
  3. added GTK+ 3 and Wayland support.
  4. added High DPI supoport under Windows.
  5. redisigned the icons by jerome KASPER <neon.king.fr[at]gmail.com>.
  6. changed some keyboard shortcuts, see Keyboard shortcuts change in wxMEdit 3.2.
  7. updated GB18030 support to GB18030-2022 standard.
  8. updated Unicode block descriptions for Unicode 15.0 (require ICU 72 or higher).
  9. fixed crash in text mode when a line exceed the limit of line length.
  10. fixed crash when pressing OK button in "About wxMEdit" dialog.
  11. fixed not responding when warping by window with text containing some tab characters.
  12. fixed broken triple click behavior in wxMEdit 3.1.
  13. fixed that the height of Information Window in MadEdit/wxMEdit (built with wxMSW-2.8) always reduce by 4.
  14. fixed fonts and encodings not restored after restart.
  15. fixed that cannot type characters via AltGr key.
  16. fixed cursor and selection rendering under Linux.
  17. fixed that IME candidate window doesn't follow the caret under Windows 10.
  18. added French translation by jerome KASPER <neon.king.fr[at]gmail.com>.
  19. 更新 - 簡体字中国語翻訳.
  20. その他マイナーチェンジとバグ修正.

3.1

  1. 追加 - 新たな特殊機能: 単語の境界 - 中国語/日本語/タイ語/ラオ語/クメール語およびビルマ文字、その他.
  2. 追加 - 新たな特殊機能: Unicode® Standard Annex #14: Unicode 改行アルゴリズムで折り返し.
  3. 追加 - 新たな機能: 列の調整.
  4. 追加 - ツールバーアイコン & ショートカットを順序付きで挿入.
  5. 変更 - 末尾行のマーク パターン.
  6. 更新 - Unicode ブロック説明を Unicode 8.0 にアップデート(ICU 56 以上が必要).
  7. 修正 - 英語以外の文字で単語を選択した時の不正確な境界.
  8. 修正 - 英語以外の文字での不正確な折り返し.
  9. 修正 - wxWidgets-3.0 GTK2 でビルドされた wxMEdit において、テンキー入力時に文字が破損する不具合.
  10. 修正 - テキストファイルの先頭、または列モードで選択範囲がゼロの時に Ctrlキー + Backspace が押された時の振る舞い.
  11. 修正 - 異なる行でダブルクリックした際に単語全体が選択されない不具合.
  12. 修正 - トリプルクリックで行全体が選択されない不具合.
  13. 修正 - 16進モードで再起動&復帰した際にファイルが保持されない不具合.
  14. 修正 - 通常テキストモードでコピーした時の改行の冗長.
  15. 更新 - 日本語翻訳 by Tilt <tiltstr[at]gmail.com>.
  16. 更新 - スペイン語翻訳 by Carlos Sánchez <carlos1994sanchez[at]gmail.com>.
  17. 更新 - 簡体字中国語翻訳.
  18. 更新 - 繁体字中国語翻訳 by Vincent_TW.
  19. 更新 - ロシア語翻訳 by Будинов Станислав <stabud[at]yandex.ru>.
  20. その他マイナーチェンジとバグ修正.

3.0

  1. 追加 - 行番号毎にブックマークを表示/印刷するオプション.
  2. 追加 - 正規表現で改行のドット一致を有効にするオプション.
  3. 追加 - 検索/置換ダイアログが可視状態でない時、ステータスバーに検索メッセージを表示する機能.
  4. 追加 - 順序付きシーケンスを挿入.
  5. 改良 - ブックマークの表示と印刷に関する動作を改善.
  6. 改良 - 空白とタブ文字の可読性 (タブ文字、および EOF マーク パターンの改善により).
  7. 修正 - 特定の折り返し設定で文字数の制限が少なくなっていた問題.
  8. 修正 - 構文の強調設定ダイアログボックス内で、更新されていない項目のリスト幅.
  9. 修正 - 任意のゼロ長アサーションでカウント/すべて置換を実行した時の無応答
  10. 更新 - Unicode ブロックの説明を Unicode 7.0 にアップデート (ICU 54 以上が必要).
  11. 追加 - ドイツ語翻訳 by Ronny Steiner <post[at]sirsteiner.de>.
  12. 追加 - ポーランド語翻訳 by Adam Massalski <massaada[at]wp.pl>.
  13. 更新 - 簡体字中国語翻訳.
  14. 更新 - 日本語翻訳 by Tilt <tiltstr[at]gmail.com>.
  15. 更新 - スペイン語翻訳 by Carlos Sánchez <carlos1994sanchez[at]gmail.com>.
  16. その他マイナーチェンジとバグ修正.

更に詳細な情報は 更新履歴 をご覧ください。
