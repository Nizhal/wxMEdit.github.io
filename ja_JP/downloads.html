更新履歴

2.9.8

  1. 追加 - 16 進モードで編集中、16 進領域に 16 進文字列を貼り付ける際の選択肢
  2. 新たに追加された機能: 16進領域への上書き貼り付け
  3. 追加 - 一般的なエンコーディングのエイリアスを使用してエンコーディング変換ダイアログボックスを表示
  4. 追加 - ツールから開かれるホームページとダウンロードページを多言語化
  5. 追加 - 中国語翻訳 zh_HK, zh_MO, zh_SG および zh に対応
  6. 追加 - Windows の %APPDATA% ディレクトリに構成設定ファイルを配置するオプション
  7. デフォルトウィンドウサイズを大きめに変更
  8. アップデートの自動確認と同じように手動アップデートで非同期的にチェック
  9. 改良 - エンコーディング グループ
  10. 修正 - clang-analyzer によるバグ検出.
  11. 修正 - Windows におけるデバッグ完了時のメモリリーク警告
  12. 更新 - FreeBSD のパッケージングをポート経由の pkgng に切り替え
  13. 更新 - 簡体字中国語翻訳
  14. 更新 - 繁体字中国語翻訳 by Vincent_TW.
  15. 更新 - スペイン語翻訳 by Carlos Sánchez <carlos1994sanchez[at]gmail.com>.
  16. 更新 - 日本語翻訳 by Tilt <tiltstr[at]gmail.com>.
  17. 更新 - ロシア語翻訳 by Shamil Bikineyev <shamilbi[at]gmail.com>.
  18. その他マイナーチェンジとバグ修正.

2.9.7

  1. 追加 - アスキーアートに適した新たなエンコーディング "CP437 Variant for ASCII-Art" に対応.
  2. 追加 - アップデートの自動確認機能.
  3. 追加 - ブックマークに対応 [mainly by Gospodin Gyurov (aka Gogo) <ggurov[at]yahoo.com> ].
  4. 追加 - 新たなコマンド: 前/次の単語を削除 (by Gogo).
  5. 追加 - 検索オプション "折り返す".
  6. 追加 - バージョン情報ダイアログ内 クレジット タブ.
  7. 追加 - オプションダイアログ内 その他 タブ.
  8. 追加 - トリプルクリックによる行選択 (mainly by cfreeer <cfreeer[at]users.sf.net>).
  9. 追加 - 保存したファイルから構文定義を更新 (mainly by cfreeer).
  10. 追加 - FreeBASIC 構文定義ファイル.
  11. 追加 - エンコーディングのグループ化.
  12. 追加 - 履歴の削除に対応.
  13. 追加 - オプションダイアログ内 16 進領域でコピーする際の選択肢.
  14. 追加 - ロシア語翻訳 by Shamil Bikineyev <shamilbi[at]gmail.com>.
  15. 改良 - Windows における最大化時の挙動 (by Gogo).
  16. 改良 - Windows におけるシステムへの統合.
  17. 改良 - 単語数カウント情報の描画
  18. 変更 - 別のエディタで次の単語を選択する際の動作.
  19. 変更 - ダイアログボックス下部バーで表示される 検索/置換 ダイアログボックスの通知メッセージ.
  20. 修正 - 置き換え時の常時折り返しなし.
  21. 修正 - Windows 中国語版においてファイル "a b" と "aÀb" を同時に開けなかった不具合.
  22. 修正 - MadEdit/wxMEdit デバッグ版でマウスキャプチャを失った時、アサーションに失敗する不具合.
  23. 修正 - wxMEdit/MadEdit でテキスト選択中に切り替えた時の不適切な振る舞い.
  24. 修正 - ファイルを開き編集領域をクリックした時の更新動作.
  25. 更新 - 簡体字中国語, 日本語, スペイン語翻訳.
  26. その他マイナーチェンジとバグ修正.

2.9.6

  1. 改良 - Mac OS X への対応.
  2. wxSmith と固定ダイアログサイズの問題により、すべてのダイアログを再設計.
  3. エンコーディング検出のリファクタリング、およびエンコーディング検出時の問題点を修正.
  4. 修正 - Windows における 16 進モード再描画時の不具合.
  5. 修正 - Unix 系 OS におけるテキスト編集領域上いくつかの行にキャレットがある状態で 検索/置換結果パネルを開いた時のクラッシュ.
  6. 修正 - ファイル関連付けを除去した時の wxWidgets エラー.
  7. 更新 - RPM, Debian および FreeBSD パッケージングファイル.
  8. 修正 - "折り返し前の最大行長" に大きな値をセットして再起動した時のクラッシュ.
  9. 改良 - ビルド構成.
  10. 更新 - スペイン語 by Carlos Sánchez <carlos1994sanchez[at]gmail.com>.
  11. 更新 - 日本語翻訳 by Tilt <tiltstr[at]gmail.com>.
  12. 更新 - 簡体字中国語翻訳.
  13. 更新 - 繁体字中国語翻訳 by Denir Li <denir.li[at]gmail.com>.
  14. その他マイナーチェンジ.
